## 💡 Core Concepts Explored
### Live Algorithmic Problem Solving
**Definition:** Timed, unassisted programming challenges designed to test logical reasoning, syntax memorization, and fundamental algorithmic knowledge in C/C++.
**Problem Solved:** It validates a developer's ability to write clean, working code from scratch without relying on external resources, simulating high-pressure coding interviews.

## 📌 Overview
This directory contains the solutions and preparation materials for the various **42 Network Exams** taken during my time at 1337.

Exams at 42 are strictly monitored, time-boxed programming challenges that test an engineer's ability to recall algorithms, build tools from scratch, and solve complex logic puzzles under pressure without internet access.
